Automated Event Invitation & Attendance System

Welcome to Our Seamless Event Management Solution

Our project is an automated event invitation and attendance management system designed to streamline the process of inviting guests to important events. Using the WhatsApp Business API, we send personalized invitations, confirm attendance, provide event details, and track guest arrivals with QR code validation.

The system ensures a smooth and efficient event experience for both the hosts and the guests, saving time and improving organization.
